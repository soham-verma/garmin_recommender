# Garmin Watch Finder

**Garmin Watch Finder** is a web application designed to help users find the **perfect Garmin watch** based on their activity needs and preferences. The app uses a comprehensive **questionnaire** to recommend the best Garmin watch suited for the user's requirements. It also includes features like a **watch catalog** and a **comparison tool** to help users make informed decisions.

---

## Features

- **Comprehensive Questionnaire**: Helps users determine the best Garmin watch based on their activities (cycling, swimming, running, etc.), feature preferences (notifications, touch, etc.), and more.
- **Watch Catalog**: Browse through a list of all available Garmin watches with detailed specifications.
- **Watch Comparison**: Compare up to 3 Garmin watches side by side to see which one fits your needs.
- **Intuitive Dashboard**: Provides an easy-to-navigate interface with links to all major features and instructions on how to use the site.

---

## Technologies Used

- **Django**: Web framework used for building the backend of the application.
- **Bootstrap**: Used for responsive design and UI components.
- **HTML5 & CSS3**: For structuring and styling the web pages.
- **Python**: The programming language used to build the project.

## Setup & Installation

### Prerequisites

Make sure you have **Python** and **pip** installed on your machine. You can check this by running:

```
python --version
pip --version
```
## Steps to Run Locally
1. Clone the repository to your local machine.
2. Run the Development Server After the dependencies are installed, run the Django development server:

   ``` python manage.py runserver ```

4. Access the Application Open your browser and go to `http://127.0.0.1:8000/` to access the application.
